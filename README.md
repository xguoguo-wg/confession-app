# ❤️ 表白页面

一个浪漫的 HTML 表白页面，部署在 GitHub Pages。

## 🌐 在线地址

**https://xguoguo-wg.github.io/confession-app/**

## ✏️ 修改内容后更新步骤

每次改完 `index.html` 后，在项目目录打开终端，执行这**三行命令**就行：

```bash
git add .
git commit -m "更新内容"
git push
```

等 1~2 分钟，网页就会自动更新。

> **记不住？** 直接复制上面三行执行即可。

## 🚀 首次部署（备忘）

```bash
git init
git add .
git commit -m "init"
git remote add origin https://github.com/xguoguo-wg/confession-app.git
git push -u origin master
```

然后在 GitHub 仓库 Settings → Pages → 选 master 分支 → Save。
